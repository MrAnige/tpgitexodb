Il vous faudra suivre ces 3 étapes:

1:Rangez le zip dans un nouveau dossier  dans xampp/htdocs/'nouveau dossier'

2: Créer votre base de données avec une ligne d'identifiant 'id' et une autre comprenant les adresses mail

Il faut que vous créez un fichier 'params.php' avec des noms de variables précises: voir exemple

$servername = "localhost";    //Nom du serveur
$dbname = "tp3";   //Correspond au nom de votre bdd
$dbuser = "yo";    //Votre nom d'utilisateur à la bdd
$dbpassword = "abcd";  //Votre mdp
